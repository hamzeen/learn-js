Overview

State of Javascript

Javascript was developed by Brendon Eich as a browser interpreted language within 2 weeks. Today it can run outside the browser thanks to Node.js which embraces JS anywhere and everywhere paradigm.

Data Types

  • Undefined | Null | Boolean| String| Symbol | Number | Object

Scopes:

  • global, local & block-scope

Path to Nodejs

V8 (Lars Bak, Danish Developer)

V8 was menat to improve performance of rendering in Chrome. The team cam up with a break through idea to convert javascript into machine code and then subsequently byte code. This was toally new got traction real fast as it drastically improved rendering speeds within Chrome.

NODE.js (Ryan Dahl)

the v8 engine paved the way for Node.js
with v8, javascript was no longer a browser interpreted language it could be compiled outside. Node.js took advantage of this & showed that JS could now be used to build servers & it could be used for server side programming. Node.js promotes anywhere & everywhere paradigm for Javascript.

Hence it enabled js to be used even for desktop applications. Ex: Electron (CodeEditor: Atom).
